git init  создает новый репозиторий Git
git add --all   добавляет изменение из рабочего каталога в раздел проиндексированных файлов
git commit -m "1commit"  подготавливает файл к отправке
git branch -M main  создаёт и даёт возможность редактировать новые ветки
git remote add origin https://github.com/maximkl1102/123.git  добавление удалёного репозитория
git push -u origin main отправка репозитория
